# Excel Sales & Order Analysis Dashboard

## Overview
Built an interactive Excel dashboard to analyse coffee order and sales data using multiple datasets.

## Tools Used
- Microsoft Excel
- Pivot Tables
- XLOOKUP / INDEX-MATCH
- Data Cleaning & Validation

## Key Features
- Combined multiple tables (orders, customers, products) using lookups
- Created pivot tables to analyse sales, revenue, and trends
- Built an interactive dashboard with slicers and charts for reporting

## Outcome
Provides a clear, high-level view of sales performance and customer trends to support reporting and decision-making.
